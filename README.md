# BookMyShow Clone

A full-stack movie ticket booking application developed using React.js and Spring Boot.

## Features
- Movie Listings
- Ticket Booking
- REST API Integration
- Responsive UI

## Tech Stack
- React.js
- Spring Boot
- Java
- MySQL

## Project Structure
book_my_show/
│── src/
│── pom.xml
│── mvnw
│── .gitignore

## Getting Started

### Backend Setup
1. Open project in IDE
2. Configure MySQL database
3. Run Spring Boot application

### Frontend Setup
```bash
npm install
npm start
